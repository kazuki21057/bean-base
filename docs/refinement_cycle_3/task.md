# 修正と改善タスク (Cycle 3)

ユーザーフィードバック（Timeの累積表示、データ読み込み不全、空項目の非表示）に対応する。特にスプレッドシートの実データ形式に対応した読み込みロジックの修正を行う。

## タスクリスト
- [x] **Phase 1: 抽出画面 (Calculator) の修正**
    - [x] `Time (s)` 列を累積時間（0スタートで加算）表示に変更
- [x] **Phase 2: データ読み込みの根本修正**
    - [x] `SheetsService` のマッピング確認・修正（スプレッドシートのヘッダー名とコードの整合性: `抽出方法`, `湯温(°C)`等）
    - [x] 日付パース (`_parseDateTime`) の強化（`yyyy/MM/dd` や `7:39` のパディング対応）
    - [x] `methodId` 等の連携キーが正しく取得できているか確認
- [x] **Phase 3: 詳細画面の改善**
    - [x] 情報がない（空文字、0、null）項目は画面に表示しないロジックを追加
- [x] **検証**
