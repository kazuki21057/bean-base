# スプレッドシート変更手順

アプリ側で豆の在庫管理機能（購入日、開封日、在庫状況など）を有効にするため、Googleスプレッドシートの **bean_master** シートに以下の列を追加してください。

## 追加する列と場所

`bean_master` シートの既存の列（豆画像URLなど）の右側に、以下のヘッダー名で新しい列を追加してください。
順序は問いませんが、管理しやすいよう末尾に追加することをお勧めします。

| ヘッダー名 | データ形式 | 入力例 | 備考 |
| :--- | :--- | :--- | :--- |
| **購入日** | 日付（yyyy/MM/dd） | `2025/12/01` | 未入力可 |
| **開封日** | 日付（yyyy/MM/dd） | `2026/01/10` | 最初の使用日 |
| **使い切り日** | 日付（yyyy/MM/dd） | `2026/01/20` | すべて使い切った日 |
| **在庫** | 真偽値 (TRUE/FALSE) | `TRUE` | 手元にある場合は `TRUE`、なければ `FALSE` または空欄 |

## ダミーデータの入力（動作確認用）

アプリでの表示確認のため、いくつかの豆に対してダミーデータを入力してください。

**例:**
- 豆A: 購入日 `2025/12/01`, 開封日 `2025/12/05`, 在庫 `TRUE`
- 豆B: 購入日 `2025/10/01`, 使い切り日 `2025/11/01`, 在庫 `FALSE` (または空欄)

※ `Apps Script` の再デプロイは**不要**です（同じURLで範囲が変わるだけなら自動で読み込まれます）。
