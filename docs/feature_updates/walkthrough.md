# 変更確認手順書 (Walkthrough)

ユーザー要望に基づくUI改善を実施しました。実機またはエミュレータで以下の項目を確認してください。

## 1. Coffee Log Edit (ログ編集)
**対象:** Log Detail Screen -> Edit Screen

1. **編集画面を開く**: 任意の抽出ログの詳細画面から編集アイコンをタップ。
2. **Score (スコア) 順序確認**:
   - [ ] "Overall" (総合評価) のスライダーが、Scoresセクションの**一番下**に配置されていることを確認してください。

## 2. Method Steps Input (メソッドステップ入力)
**対象:** Method Detail Screen -> Edit Mode

1. **メソッド編集**: 任意のメソッドを開き、編集モードに切り替えてください。
2. **Pouring Steps (注湯ステップ)**:
   - [ ] **End Time** の入力欄が `mm:ss` (例: `1:30`) 形式で表示されていることを確認してください。
   - [ ] 値を変更（例: `1:45`）して保存し、再度開いたときに正しく `1:45` と表示されるか確認してください。
   - [ ] 入力した時間に基づいて、内部の秒数が正しく計算されているか（保存後のデータ整合性）を確認してください。

## 3. Master Grid Images (画像表示)
**対象:** Dashboard (Home), Master Lists

1. **Inventory (Home)**:
   - [ ] 在庫リストの豆画像が表示されているか確認してください。
   - [ ] Google DriveのURLなどが登録されている場合でも、画像が表示されるよう改善しました（プレースホルダーではなくなるはずです）。
2. **Master Tabs**:
   - [ ] **Beans, Grinders, Drippers, Filters** の各タブで、登録された画像が正しく表示されているか確認してください。
   - [ ] 画像がないアイテムのみ、アイコンのプレースホルダーが表示されます。

## 検証
以下のコマンドですべてのテストがパスすることを確認済みです。
```bash
flutter test
```
