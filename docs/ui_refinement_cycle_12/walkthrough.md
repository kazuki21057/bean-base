# データ可視化改善 (Cycle 12) ウォークスルー

## 1. 変更内容
### Radar Chart (レーダーチャート)
- **スケール固定**: チャートが常に **0〜10点** の範囲で描画されるようになりました。
- **グリッド線**: 2点刻み (2, 4, 6, 8, 10) のグリッド線を表示し、各項目のスコアが直感的に分かりやすくなりました。

### PCA Scatter Plot (主成分分析プロット)
- **スコア連動スタイル**:
  - **色**: `Light Blue` (低得点) 〜 `Red` (高得点) のグラデーションで表示されます。
  - **サイズ**: 点の大きさがスコアに比例して変化します（高得点ほど大きい）。
- **凡例 (Legend)**:
  - プロットの下に「Low Score -> High Score」のカラーバーと、サイズの目安を表示しました。

## 2. 検証手順
1. **アプリ起動**:
   ```bash
   flutter run -d chrome
   ```
2. **Statistics画面へ移動**:
   - 下部タブまたはサイドバーから「Stats」を選択。
3. **Radar Chart確認**:
   - チャートの外枠が10.0（最大）であることを確認。
   - グリッド線が等間隔に5本あることを確認。
4. **PCA Plot確認**:
   - 自分の記録した豆がプロットされていることを確認。
   - **高得点（8-10点）** の豆が「赤く」「大きく」表示されているか確認。
   - **低得点（5-6点）** の豆が「青く」「小さく」表示されているか確認。
