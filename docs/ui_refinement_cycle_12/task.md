# データ可視化の改善 (Cycle 12)

## 目的
Stats画面のデータ可視化を改善し、直感的で正しい理解を促進する。
1. **Radar Chart**: スケールを0-10に固定し、軸の視認性を向上させる。
2.  **PCA Scatter Plot**: スコアに応じて点の色（青→赤）とサイズ（小→大）を変え、高評価の豆を強調する。
