# BeanBase 2.0

BeanBase 2.0 is a comprehensive coffee bean management and brewing logging application.

## Features

- **Master Data Management**: Manage coffee beans, equipment, and other master data.
- **Brewing Calculator**: Calculate brewing ratios and parameters.
- **Data Logging**: Log your coffee brewing sessions and track your history.
- **Dashboard**: Visualize your coffee consumption and brewing stats.

## Getting Started

This project is a Flutter application.

### Prerequisites

- Flutter SDK
- Dart SDK

### Installation

1. Clone the repository:
   ```bash
   git clone git@github.com:kazuki21057/bean-base.git
   ```
2. Install dependencies:
   ```bash
   flutter pub get
   ```
3. Run the app:
   ```bash
   flutter run
   ```

## Project Structure

- `lib/`: Contains the source code for the Flutter application.
- `docs/`: Documentation files.
- `original-data/`: Original data files (if any).
